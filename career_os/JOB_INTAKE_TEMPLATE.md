# Job Intake Template

```markdown
- Job ID:
- Source:
- URL:
- Company:
- Job title:
- Location:
- Remote/hybrid/onsite:
- Salary range:
- Employment type:
- Job description:
- Required qualifications:
- Preferred qualifications:
- Tools mentioned:
- Healthcare requirements:
- Certifications required:
- Years of experience:
- Application deadline:
- Application complexity: Easy / Standard / Complex
- Easy Apply? Yes/No
- Requires account? Yes/No
- Requires sensitive/legal answers? Yes/No
- Digital Juan score:
- Council verdict:
- Resume version:
- GitHub project to feature:
- Tailoring notes:
- Final status:
```
