# Weekly Application OS

## Weekly Plan

| Day | Focus | Output |
| --- | --- | --- |
| Monday | Collect 25 jobs, dedupe, score. | Top Apply/Maybe/Skip list. |
| Tuesday | Apply to top 5. | Five applications or manual-final-step packets. |
| Wednesday | Apply to next 5 and send recruiter messages. | Five more applications plus outreach. |
| Thursday | Follow-ups and interview prep. | Five follow-ups and one STAR story practice. |
| Friday | Pipeline review and salary strategy. | Status updates, weak-role cleanup, next searches. |
| Weekend | One portfolio improvement or skill refresh only if it directly helps applications. | One practical improvement. |

## Weekly Minimum

- 10 quality applications.
- 5 recruiter messages.
- 5 follow-ups.
- 1 interview prep session.
- 1 portfolio improvement only if it directly helps applications.

## Review Metrics

- Roles found.
- Roles scored 75+.
- Applications submitted.
- Manual-final-step queue count.
- Follow-ups due.
- Interviews.
- Average score of applications submitted.
- Salary clarity rate.
